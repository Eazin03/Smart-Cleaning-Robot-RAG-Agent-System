# 扫地机器人&扫拖一体机 完整版维护保养手册
## 一、日常基础日常保养（每日/每次使用后）
1. 污水箱处理
每次拖地完成后，及时倒掉污水，避免污水长期留存滋生细菌、产生臭味、发霉发黏。
2. 拖布清洁
自清洁机型虽自动洗拖布，但潮湿梅雨季建议开启烘干；无自清洁机型，每次用完拆下手洗、完全阴干。
3. 外观简单擦拭
用干燥软布擦拭机身外壳、防撞条，去除表面灰尘油污，保持机身散热通畅。
4. 地面简单整理
清理长线、塑料袋、大块杂物，减少机器卡困、缠绕概率，延长配件寿命。

## 二、短期保养（3–5天/每周）
1. 尘盒清理
倒掉盒内垃圾，擦拭尘盒内壁，可水洗款清水冲洗，完全晾干再装机。
2. 滚刷深度清理
剪开缠绕长发、宠物毛发、棉线，清理两端轴承缝隙杂物，防止滚刷卡死、电机过载。
3. 边刷维护
取下边刷底座，清理缠绕垃圾，检查刷毛是否歪斜、磨损。
4. 传感器清洁
激光雷达、悬崖传感器、避障摄像头、充电触点，用干棉签轻轻除尘，保证定位、避障、回充精准。
5. 行走轮清理
清理驱动轮、万向轮夹缝毛发灰尘，防止行走打滑、原地打转、走位偏移。

## 三、中期保养（每月）
1. 滤网保养
可水洗滤网：清水轻冲、阴干24小时，禁止暴晒；不可水洗滤网：拍打除尘，堵塞严重直接更换。
2. 水箱深度清洁
清水箱、污水箱内壁冲洗，定期用温和柠檬酸溶液浸泡除垢、去味、抑菌。
3. 基站底盘清洁
全能基站刷洗盘、污水槽、排水滤网，清理残留污渍、污垢，避免发霉异味。
4. 地图校准
家具大范围移动后，重置地图重新建图，保证清扫路线合理、不漏扫。
5. 固件检查
连接APP检查系统版本，及时升级优化算法、避障、续航逻辑。

## 四、长期深度保养（季度/半年）
1. 耗材定期更换
边刷：3–6个月
拖布：3–6个月
HEPA滤网：6–12个月
集尘袋：1–2个月（养宠缩短周期）
2. 电池养护
避免长期深度亏电存放，长期不用保持50%电量，放置干燥通风处。
3. 风道与风机除尘
清理进风口、风道积灰，防止吸力下降、噪音变大、散热不良。
4. 机身接口防潮
潮湿地区定期检查充电接口、触点，擦干水渍，防止氧化接触不良。

## 五、禁忌操作（禁止做的事）
1. 主机整机水洗、浸泡、淋雨，会直接烧坏电机与主板。
2. 使用84消毒液、强酸强碱、高泡清洁剂加入水箱，腐蚀管路与拖布。
3. 用酒精、消毒液直接擦拭雷达、镜头镀膜，造成永久模糊。
4. 长期不倒污水、不烘干拖布，引发霉菌、异味、蚊虫滋生。
5. 强行拖拽机器、暴力掰扯滚刷，造成齿轮、卡扣损坏。
6. 高温暴晒、靠近暖气、明火摆放，加速电池老化。

## 六、分场景专属保养要点
1. 养宠家庭
每次使用必清滚刷毛发，缩短滤网、尘袋更换周期，常开除菌、烘干功能。
2. 木地板家庭
严控水箱出水量，定期抬升拖布避开木质软装，禁止积水滞留。
3. 南方潮湿地区
强制开启拖布烘干，水箱勤清洗，基站定期除霉抑菌。
4. 多楼层复式
固定基站位置，不乱搬机器，定期切换校准多楼层地图。

## 七、故障预防保养小技巧
1. 定期轻柔除尘传感器，可减少90%迷路、乱撞、避障失灵问题。
2. 坚持烘干拖布，杜绝拖布发臭、地板异味。
3. 长期插电待机，智能补电，电池寿命更长久。
4. 定期分区清扫+深度全屋清扫结合，减轻机器负荷。
